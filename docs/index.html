OpenEASM Alpha

External Attack Surface Management public non intrusif

Auditez l’exposition Internet d’un domaine grâce à des contrôles publics : DNS, messagerie, TLS/SSL, headers web, sous-domaines, inventaire IP, CTI légère et scoring de risque.

Voir le projet GitHub Installation rapide

Pourquoi OpenEASM ?

Cartographie d’exposition

Découverte des sous-domaines, IP publiques, services web et signaux DNS.

Analyse de risque

Score exécutif par piliers : DNS, messagerie, web, TLS/SSL, surface exposée, CTI et CVE passives.

Reporting

Exports PDF, Excel et JSON pour faciliter la restitution technique et décisionnelle.

Contrôles réalisés

DNS public
SPF / DMARC / MX
TLS/SSL
Headers HTTP
Sous-domaines passifs
Inventaire IP
CTI légère / DNSBL
CVE passives
Score exécutif
Rapports PDF / Excel / JSON

Usage responsable

OpenEASM Alpha est conçu pour un usage défensif et autorisé. L’outil ne réalise pas de scan de ports agressif, pas d’exploitation, pas de bruteforce et pas de test destructif.

Installation rapide

git clone https://github.com/Swapshadow/open-easm.git
cd open-easm
docker-compose up -d --build

http://localhost:8000

Open source

Le projet est publié sur GitHub afin de permettre l’amélioration continue, les retours communautaires et l’évolution vers une solution EASM libre, simple et compréhensible.
