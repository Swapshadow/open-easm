OpenEASM

Plateforme open source d’External Attack Surface Management

Voir le projet GitHub

Présentation

OpenEASM est un outil permettant d’analyser l’exposition externe d’une organisation : domaines, ports ouverts, services exposés, certificats TLS, technologies détectées, risques et rapports.

Cartographie d’exposition

Inventaire des domaines, sous-domaines, services et ports exposés sur Internet.

Analyse de risque

Score de risque, détection de mauvaises configurations et priorisation des actions.

Reporting

Génération de rapports techniques et exécutifs pour la DSI, RSSI ou direction.

Installation rapide

git clone https://github.com/Swapshadow/open-easm.git
cd open-easm
docker compose up -d --build

Objectif du projet

Fournir une solution simple, lisible et exploitable pour surveiller la surface d’attaque externe d’une organisation et faciliter la mise en conformité cyber.
