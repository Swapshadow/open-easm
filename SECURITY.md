# Politique de sécurité

OpenEASM Alpha est un outil défensif public non intrusif.

Pour signaler un problème de sécurité dans le projet, ouvrez une issue GitHub sans divulguer d’informations sensibles, ou contactez directement le mainteneur du dépôt.

Ne publiez pas de secrets, tokens, mots de passe ou données d’audit sensibles dans une issue publique.
